== Description ==

= Like Box Widget & Shortcode - Social Profile =

A very easy and simple facebook like box widget & shortcode plugin for wordPress websites/blogs.

You can promote any page with like box. It is also responsive so you can use it on any page and post.

You can attract visitor to your page with like box and divert your site trafic to your page.

To activate like box widget on your website. Go to widgets area in admin dashboard, find the "Like box Widget" named widget and activate in any widget of theme.

For pages and groups you can choose the user roles whose posts will be shared- page owner/ group admin, group member/page fans or both. Hashtags and mentions included in posts will be preserved. The page plugin comes with a number of great features and functionality to allow you create anything from free simple feeds to custom feeds, including group events, like box, feed gallery and more to increase engagement of your fans.

**Major features in Like Box plugin include:**

* Like box Widget
* Like box Shortcode
* Custom Application ID
* Custom Widget Title
* Widget Custom Width (Min Width: 180 - Max Width: 500)
* Widget Custom Height (Min Height: 70 )
* Full Width Widget
* Show Page Cover
* Show Page Header
* Small And Large Page Header
* Show Friends
* Show Page Post/Stream
* 43 Language support for like box


== Installation ==
Install like box Widget & Shortcode either via the WordPress.org plugin directory or by uploading the files to your server.

After activating Install like box Widget & Shortcode, go to widgets menu under appearance menu.

Find the widget named with 'like box Widget and shortcode'.

Now drag and drop the widget in available widget areas into your theme.

Finally, configure your widget setting and save.

That's it. You're ready to go!
